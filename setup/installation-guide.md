# ⚙️ Installation Guide

## Install Wireshark

1. Download from official website:
   https://www.wireshark.org/

2. During installation:

   * Install Npcap (Required)
   * Enable packet capture permissions

---

## Verify Installation

Open Wireshark and check:

* Network interfaces visible
* Able to start capture

---

## Start Capture

1. Select active interface (Wi-Fi/Ethernet)
2. Click Start
3. Generate traffic:

   * Open browser
   * Visit websites
4. Stop capture after 2–3 minutes
5. Save file as 'traffic_capture'.
